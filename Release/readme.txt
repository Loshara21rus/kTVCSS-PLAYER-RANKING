1. Необходимо развернуть MySQL базу kTVCSS для работы статистики. Выполнить содержимое скрипта в папке MySQL на своем сервере.
2. Из папки GameServer необходимо залить плагины и отредактировать конфиги на вашем сервере на основе содержимого в конфигах у меня. Также необходимо в корневой директории вашего игрового сервера создать папку statistics.
3. Развертывание ПО на вашем сервере из папки Windows Server. Для корректной работы нужно собрать ProcessKeeper из гита.
